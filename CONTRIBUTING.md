
# Contributing

Any and all contributions to Lambda Mountain are dutifully appreciated. All submissions are assumed to grant permission to use under the [permissive MIT license](https://github.com/Lambda-Mountain-Compiler-Backend/lambda-mountain/LICENSE).

To see open problems, see the [issue tracker](https://github.com/Lambda-Mountain-Compiler-Backend/lambda-mountain/issues). For suggestions or questions, open a new issue.
